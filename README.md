# CertDirectory Public Documents & Agreements

This repository serves as the official, transparent record for CertDirectory's public commitments, formal agreements, and significant announcements related to our platform and community partnerships.

## Key Documents

* **[CNCF Community Commitment Letter](./docs/CNCF_Commitment_Letter.pdf)**: Our formal pledge providing the CertDirectory Professional Tier free of charge, in perpetuity, to all current and future CNCF Community Groups.

## Repository Scope

We use this space to maintain permanent records of:

* Public partnership agreements and commitments (such as our work with the CNCF).
* Major policy announcements affecting community tiers.
* Open-source program definitions.

For more information about the platform, visit [credentials.certdirectory.io](https://credentials.certdirectory.io).
